# 💊 PharmaAI - Pharmacy Management System

PharmaAI is a modern pharmacy management application designed to simplify the daily operations of a pharmacy through an intuitive dashboard and intelligent management tools.

The application allows pharmacists to manage medicines, patients, sales, inventory, and monitor business activities in real time.

## ✨ Features

### 📊 Dashboard
- Real-time statistics
- Daily sales monitoring
- Revenue tracking
- Critical stock alerts
- Inventory value calculation
- Availability percentage

### 💊 Medicine Management
- Add medicines
- Update medicines
- Delete medicines
- Search medicines
- Categorize medicines
- Manage expiration dates

### 👥 Patient Management
- Register patients
- Update patient information
- Search patients

### 🛒 Sales Management
- Create new sales
- View sales history
- Calculate totals automatically

### 📦 Stock Management
- Inventory monitoring
- Low stock alerts
- Expiration alerts

### 🤖 AI Assistant
- AI integration to assist pharmacists

### 🔐 Security
- Authentication with Spring Security
- Protected API endpoints
